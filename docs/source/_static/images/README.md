# Images Directory

Place your tutorial images in this directory. The following images are referenced in the documentation:

- `cnn_architecture.png` - Overall CNN architecture diagram
- `convolution_operation.png` - Convolution operation visualization
- `relu_activation.png` - ReLU activation function diagram
- `pooling_operation.png` - Max/average pooling visualization

## Image Guidelines

- Use PNG format for diagrams and figures
- Recommended width: 800-1200 pixels for full-width images
- Use descriptive filenames
- Optimize file size when possible
